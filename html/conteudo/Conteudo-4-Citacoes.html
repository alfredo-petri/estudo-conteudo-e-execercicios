Citações


Abaixo serão exibidas as principais tags utilizadas para fazer citações de textos em html


A tag HTML <blockquote> define uma seção citada de outra fonte. Os navegadores geralmente recuam tags <blockquote>. Ex:

Este texto possui a tag <blockquote> em sua formatação

A tag HTML <q> define uma citação curta.Os navegadores normalmente inserem aspas ao redor da citação. Ex: Este texto possui a tag <q> em sua formatação

A tag HTML <abbr> define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". A marcação de abreviaturas pode fornecer informações úteis aos navegadores, sistemas de tradução e motores de busca. Dica: Use o atributo global title para mostrar a descrição da abreviatura/acrônimo ao passar o mouse sobre a tag. Ex: Este texto possui a tag <abbr> em sua formatação = A linguagem HTML surgiu na década de 1990.

A tag HTML <address> define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc. O texto contido na tag <address> geralmente é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois da tag <address>. Ex: Este texto possui a tag <address> em sua formatação =

Written by John Doe. Visit us at:Example.com - Box 564, Disneyland - USA

A tag HTML <cite> define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.). O texto contido na tag <cite> geralmente é renderizado em itálico. Dica: O nome de uma pessoa não é o título de uma obra. Ex: Este texto possui a tag <cite> em sua formatação = The Scream by Edvard Munch. Painted in 1893.

A tag <bdo> é usada para substituir a direção atual do texto. Ex: Este texto possui a tag <bdo> em sua formatação = This text will be written from right to left
