Formulário Basico

Definição

Um formulário HTML é usado para coletar dados do usuário. As informações geralmente são enviadas a um servidor para processamento.

A tag HTML <form> é usada para criar um formulário HTML para coletar os dados.

A tag <form> é um contêiner para diferentes tipos de elementos de entrada, como: campos de texto, caixas de seleção, botões de opção, botões de envio, etc. Todos os diferentes elementos do formulário são abordados nesta página: w3schools - Atributos

Tags Relacionadas

Input

A tag HTML <input> é o elemento de formulário mais usado. Uma tag <<input> pode ser exibida de várias maneiras, dependendo do atributo type.

Label

A tag html <label> define um rótulo para muitos elementos do formulário.

O elemento <label> é útil para usuários de leitores de tela, porque o leitor de tela lerá o rótulo em voz alta quando o usuário focar no elemento de entrada. A tag <label> também ajuda usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de opção ou caixas de seleção) - porque quando o usuário clica no texto dentro do elemento <label>, ele alterna o botão de opção/caixa de seleção.

Dica: O atributo for da tag <label> deve ser igual ao atributo id do elemento <input> para uni-los.

Atributos

Action

O atributo action é utilizado junto a tag <form> e define a ação a ser executada quando o formulário for enviado. Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.

For

O atributo for é utilizado junto a tag <label>. O atributo for faz um tipo de link entre a tag <label> e a tag <input>. Logo o for tem que apontar para o id do <input>.

Id

O atributo id é utilizado junto a tag <input> para criar um identificador exclusivo para aquela tag. Além de o mesmo fazer o link entre as tags <label> e <input> ele também é utilizado para aplicar estilos css.

Type

O atributo type é utilizado junto a tag <input>, ele identifica o tipo de <input> que esta sendo usado naquele campo do formulario.

Name

O atributo name é utilizado junto a tag <input>. Sempre que realizamos um envio de dados através de um formulário, o atributo name é o identificador dos dados preenchidos pelo usuário que serão entregues ao servidor.

Placeholder

O atributo placeholder é utilizado junto a tag <input>.

O atributo placeholder especifica uma breve dica que descreve o valor esperado de um campo de entrada (uma breve descrição do formato esperado). A dica curta é exibida no campo de entrada antes que o usuário insira um valor.
