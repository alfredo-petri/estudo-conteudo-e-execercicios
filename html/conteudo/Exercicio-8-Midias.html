Imagem

Definição

As imagens podem melhorar o design e a aparência de uma página da web. A tag responsável por inserir uma imagem na página é <img>

Atributos

Os principais atributos relacionados a tag <img> são os atributos: src e alt

O atributo src especifica o caminho onde a imagem está localizada

O atributo alt especifica um texto altenativo que será exibido quando por algum motivo a imagem não puder ser carregada

Fav-Icon

Definição

Um favicon é uma pequena imagem exibida ao lado do título da página na guia do navegador. A tag html reponsavel por adicionar um Fav-Icon é a tag <link>, esta deve ser inserida dentro da tag <head> junto com as demais metatags

Dica: Um favicon é uma imagem pequena, portanto deve ser uma imagem simples com alto contraste.

Atributos

Os principais atributos relacionados a tag <link> são os atributos: rel; type e src.

O atributo rel especifica o relacionamento entre o documento atual e o documento/recurso vinculado. O valor deste atributo quando usado em um Fav-Icon deve ser = icon

O atributo type especifica o tipo de mídia do documento/recurso vinculado. O valor deste atributo quando usado em um Fav-Icon deve ser = image/x-icon

O atributo src especifica o caminho onde a imagem está localizada.

Video

Definição

A tag HTML <video> é usada para mostrar um video em uma página da web.

Atributos

Os principais atributos relacionados a tag <video> são os atributos: source; src e control

O atributo source é usado para especificar mais de um recurso de mídia para elementos de mídia. Ele permite que você especifique arquivos alternativos de video/áudio/imagem que o navegador pode escolher, com base no suporte do navegador ou na largura da janela de visualização. O navegador escolherá o primeiro <source> que suportar.

O atributo src especifica o caminho onde o video está localizado. este é obrigatório quando o atributo <source> for usado.

O atributo controls O atributo controls adiciona controles de video, como reproduzir, pausar e volume

O atributo autoplay O atributo autoplay faz com que o video comece a ser reproduzido assim que o mesmo for carregado. Dica: o muted pode ser utilizado em conjunto com o atributo autoplay para que assim o video inicie automáticamente mutado.

Dica: para uma melhor vizualização da página enquanto ela é carregada é necessário definir a largura e a altura do player de video através dos atributos width e height

Iframe

Definição

Um iframe HTML é usado para exibir uma página da web dentro de uma página da web. A tag HTML <iframe> especifica um quadro embutido. Um quadro embutido é usado para incorporar outro documento no documento HTML atual.

Atributos

Os principais atributos relacionados a tag <iframe> são os atributos: title; src; e control

O atributo src especifica o caminho onde está localizado o conteúdo que será exibido no iframe.

É uma boa prática sempre incluir um atributo title para o <iframe>. Isso é usado por leitores de tela para ler qual é o conteúdo do <iframe>.

Os atributo width e height. O atributo width especifica a largura do <iframe> já o atributo height especifica a altura.
