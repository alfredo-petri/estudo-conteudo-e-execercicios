Tabelas


Definição

Uma tabela em HTML consiste em um conjunto de células dentro de linhas e colunas

Atributos

table

A tag html <table> define que será inserida uma estrutura de tabela a seguir

tr

A tag <tr> define uma linha em uma tabela HTML. Esta tag contém uma ou mais tags <th> ou <td>.

th

A tag <th> define um cabeçalho para uma coluna de uma tabela

td

A tag html <td> define o conteudo de uma célula de dados padrão em uma tabela HTML

caption

A tag html <caption> define uma legenda para a tabela.

colgroup

A tag html <colgroup> especifica um grupo de uma ou mais colunas em uma tabela para formatação.

col

A tag html <col> especifica propriedades de coluna para cada coluna dentro de uma tag <colgroup>.

thead

A tag é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML. A tag html <thead> é usado em conjunto com os elementos <tbody> e <tfoot> para especificar cada parte de uma tabela (cabeçalho, corpo, rodapé).

tbody

A tag é usada para agrupar o conteúdo do corpo em uma tabela HTML. A tag html <tbody> é usado em conjunto com os elementos <thead> e <tfoot> para especificar cada parte de uma tabela (cabeçalho, corpo, rodapé).

tfoot

A tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML. A tag html <tfoot> é usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma tabela (cabeçalho, corpo, rodapé).
