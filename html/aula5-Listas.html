Listas


As listas html permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.


Existem 3 tipos de listas html:


Listas não Ordenadas <ul>

Listas não ordenadas são utilizadas quando a ordem dos itens não importa. Uma lista não ordenada começa com a tag <ul>. Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão. Cada item da lista começa com a tag <li>.

Ex: Lista não ordenada = Meus Hobbies

  • Brincar com o meu filho
  • Assistir: Animes, Filmes
  • Ler: Livros, HQs, Mangás
  • Jogar RPG de mesa, Jogos Online, Jogos de tabuleiro
  • Praticar atividade fisica: Pedalar, Nadar, Jogar basquete...

Listas Ordenadas <ol>

Listas ordenadas são utilizadas quando a ordem dos itens importa, como por exemplo em um ranking. Uma lista ordenada começa com a tag <ol>. Os itens da lista serão marcados com números por padrão. Cada item da lista começa com a tag <li>.

Ex: Lista ordenada = Minhas Bebidas Preferidas

  1. Água
  2. Suco Natural (Limão, Laranja, Acerola...)
  3. Drinks
  4. Cerveja
  5. Destilados (Vinho, Tequila...)

Listas de Descrição <dl>

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo. A tag <dl> define a lista de descrição, a tag <dt> define o termo (nome) e a tag <dd> descreve cada termo.

Ex: Lista de descrição = Termos da linguagem HTML

html
Linguagem de Marcação de Hipertexto
head
A tag <head> é usada como um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a tag <body>
body
A tag <body> contém todo o conteúdo de um documento HTML, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc

Listas Combinadas

Os tipos de lista podem ser combinados de acordo com a necessidade do usuário.

Ex: Lista combinada: Meus Hobbies

  • Brincar com o meu filho
  • Assistir:
    1. Animes
    2. Séries
    3. Filmes
  • Ler:
    1. Mangás
    2. Livros
    3. HQs
  • Jogar:
    1. RPG
    2. Jogos Online
    3. Jogos de Tabuleiro
  • Praticar atividade física:
    1. Basquete
    2. Natação
    3. Musculação
    4. Pedalar
