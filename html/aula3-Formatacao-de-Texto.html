Abaixo serão exibidos as principais tags utilizadas para formatar textos em html


A tag HTML b define texto em negrito, sem qualquer importância extra. Ex: Este texto possui A tag bold em sua formatação

A tag HTML strong define um texto de grande importância. O conteúdo interno normalmente é exibido em negrito. Ex: Este texto possui a tag strong em sua formatação

A tag HTML i este costuma ser usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, o nome de um navio, etc.. O conteúdo interno normalmente é exibido em itálico. Ex: Este texto possui a tag i em sua formatação

A tag HTML em define o texto como enfatizado. O conteúdo interno normalmente é exibido em itálico. Um leitor de tela pronunciará as palavras com ênfase, usando ênfase verbal. Ex: Este texto possui a tag em em sua formatação

A tag HTML mark define o texto que deve ser marcado ou destacado. Ex: Este texto possui a tag mark em sua formatação

A tag HTML del define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha no texto excluído. Ex: Este texto possui a tag del em sua formatação

A tag HTML ins define um texto que foi inserido em um documento. Os navegadores geralmente sublinharão o texto inserido. Ex: Este texto possui a tag ins em sua formatação

A tag HTML small define um texto menor. Ex: Este texto possui a tag small em sua formatação

A tag HTML sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas. Ex: Este texto possui a tag sub em sua formatação = H 2 O

A tag HTML sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé. Ex: Este texto possui a tag sup em sua formatação = WWW1
